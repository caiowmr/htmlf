  • tag EM deixa o texto com enfase
  • Tag b deixa o texto em negrito
  • A tag STRONG diz que é um texto importante
  • A tag u deixa o texto sublinhado
  • a tag INS formata o texto para um inserir
  • a tag DEL indica que o texto foi deletado
  • A tag mark deixa o texto marcado
  • A tag small deixa o texto pequeno
  • A tag sup deixa o texto sobrescrito
  • A tag sub deixa o texto subscrito
  • A tag q deixa o texto entra aspas
